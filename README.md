# Finance Dashboard Backend API

## Project Overview
A RESTful API for managing financial records with role-based access control. Built with Node.js, Express, and JWT authentication.

## Features Implemented
- ✅ User Authentication (JWT based)
- ✅ Role-Based Access Control (Admin, Analyst, Viewer)
- ✅ Financial Records CRUD Operations
- ✅ Dashboard Analytics (Income, Expenses, Net Balance)
- ✅ Category-wise Breakdown
- ✅ Filtering by date, category, and type
- ✅ Input Validation & Error Handling

## Tech Stacks
- Node.js
- Express.js
- JWT for authentication
- In-memory data storage
